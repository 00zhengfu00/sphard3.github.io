# 修改Vultr服务器密码
详解怎么修改[Vultr](https://www.vultr.com/?ref=7295225)服务器 root 用户密码

先通过命令行连接上服务器，然后使用命令：`passwd root`<br>
然后根据提示输入新的密码<br>
修改完成之后立即生效(请谨慎操作)，断开服务器，下次再连接服务器就需要输入修改后的新密码了
